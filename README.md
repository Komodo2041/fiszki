## Funkcje
- Kliknij fiszkę → odwróć (front/back)
- Losowe słówka z listy
- Prosty interfejs do nauki

## Tech stack
- **Vue.js 3** (Composition API lub Options API)
- **HTML/CSS** (responsywny design)

## Jak uruchomić lokalnie
1. `git clone https://github.com/Komodo2041/fiszki.git`
2. Otwórz `index.html` w przeglądarce
